Please note that this bootstrapper is an early build, which means it may contain unexpected bugs. It is advised not to modify PyLan's contents, as this may break the bootstrapper. Additionally, you will need internet access to run the bootstrapper. If you encounter any bugs, please report them by creating a new issue on our GitHub page: https://github.com/User000K/PyLan-Public. Alternatively, you can manually install PyLan from the same GitHub repository. Individual DLLs will also be provided in case you are missing any.
